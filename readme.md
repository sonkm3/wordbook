# A study for django and nuxtjs with making flashcard app.
## backend
django + django rest framework

`cd wordbook-backend`  
`pip install -r requirements.txt`  
`./manage.py makemigrations`  
`./manage.py migrate`  
`./manage.py runserver`

## frontend
nuxtjs + vuetify

`cd wordbook-frontend`  
`npm -i`  
`cd wordbook`  
`npm -i`  
`npm run dev`

open `http://localhost:3000/`
