# a practice for django and nuxtjs by making flashcard app.
## backend
django + django rest framework

`pip install -r requirements.txt`  
`./manage.py makemigrations`  
`./manage.py migrate`  
`./manage.py runserver`

## frontend
nuxtjs + vuetify

`npm -i`  
`cd wordbook`  
`npm -i`  
`npm dev run`

open `http://localhost:3000/`
